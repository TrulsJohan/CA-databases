# CA-databases

##Endpoints

PUSH make a user:
http://localhost:3000/register

GET all movies:
http://localhost:3000/movies/

GET movie by user:
http://localhost:3000/movies/user_id

PUSH a new movie:
http://localhost:3000/movies/user_id


##Optional endpoints:

PUT update movie:
http://localhost:3000/movies/movie_id

DELETE delete movie:
http://localhost:3000/movies/movie_id
